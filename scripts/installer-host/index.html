OmniKey AI

Self-hosted bring-your-own-key (BYOK) wrapper for top AI models.

Install

macOS / Linux

curl -fsSL https://omnikeyai.ca/install.sh | bash

Windows (PowerShell)

iwr -useb https://omnikeyai.ca/install.ps1 | iex

Requires Node.js 18+. The script installs the omnikey-ai CLI, runs onboarding, starts the daemon, and downloads & opens the matching desktop installer for your platform.
